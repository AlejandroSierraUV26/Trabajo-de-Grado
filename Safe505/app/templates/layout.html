Información del Proyecto
Propósito de la Herramienta

Esta aplicación fue desarrollada como proyecto de tesis con el objetivo de evaluar automáticamente la postura de seguridad de Gestión de la Postura de Seguridad en la Nube (CSPM) de un entorno AWS, tomando como base las recomendaciones del estándar CIS AWS Foundations Benchmark.

Manual Básico de Uso
  • Dashboard General: Muestra un resumen ejecutivo de las vulnerabilidades encontradas y la mejora histórica tras la aplicación de medidas correctivas.
  • Ejecutar Escaneo: Utiliza el botón amarillo en el Dashboard para lanzar una nueva evaluación mediante Boto3 (Python) contra tu cuenta de AWS.
  • Módulos de Infraestructura: Navega por S3, EC2, RDS, etc., en el menú lateral para ver el detalle técnico y los recursos específicos que fallan los controles.
  • Remediación: Haz clic en cualquier control marcado como FAILED para ver las instrucciones de mitigación.
